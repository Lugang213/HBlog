确认完毕后，程序将生成一对密钥存放在以下文件夹：/users/用户/.ssh/

私钥保存在您的电脑上，公钥交项目负责人添加到服务器上。用户必须拥有与服务器公钥所配对的私钥，才能访问服务器上的代码库。
