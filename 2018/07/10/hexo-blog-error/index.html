GitHub中使用hexo搭建博客遇到的报错处理

Jul 10, 2018 | GitHub
条评论

1. GitHub访问失去CSS样式

  • 原因： 样式加载不出来，会报404错误，URL地址不对。
  • 解决： 修改_config.yml文件中的url地址和根目录

2. 自己配置域名后CSS样式不显示

  • 原因： 404错误，发现URL下多了一层文件夹
  • 解决： 修改_config.yml文件中的url地址和根目录
