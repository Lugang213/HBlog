Git分支管理

Oct 16, 2017 | Git
条评论

分支操作允许创建另一路线，从开发主线上分离开来，然后在不影响主线的同时继续工作。 例如，我们发布了1.0版本的产品，可能需要创建一个分支，以便将2.0功能的开发与1.0版本中错误修复分开。
